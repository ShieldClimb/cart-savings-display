Cart Savings Display for WooCommerce shows customers their total savings in the cart & checkout, boosting conversions and purchase confidence.

== Description ==
Boost your WooCommerce store’s conversions by displaying customers’ total savings in the cart and checkout. 

Cart Savings Display for WooCommerce helps build trust and encourages customers to complete their purchases by clearly showing how much they are saving on discounts, promotions, and sale prices.

See more via [this website](https://shieldclimb.com/free-woocommerce-plugins/cart-savings-display/).

== Installation ==

1. Download our plugin from https://shieldclimb.com/free-woocommerce-plugins/cart-savings-display//
2. At your WordPress admin, Go to Plugins > Add New Plugin > Upload Plugin > Choose File (Choose the file you downloaded just now) > Click Install
3. Click Activate
